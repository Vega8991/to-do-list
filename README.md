Una aplicación web de gestion de tareas. Cada tarea es una misión que completar, haciendo la productividad más divertida y visual.

## Características principales

- **Agregar misiones**: Crea nuevas tareas con un simple input y botón
- **Listar tareas**: Visualiza todas tus tareas activas en tiempo real
- **Completar misiones**: Marca tareas como completadas con efecto visual (tachado y cambio de color)
- **Eliminar tareas**: Borra tareas que ya no necesitas
- **Persistencia de datos**: Usa LocalStorage para mantener tus tareas al recargar la página
- **Estadísticas en vivo**: Contador de tareas completadas y activas
- **Diseño responsive**: Adaptado para móviles, tablets y escritorio
